Social Media Image Generator 1 Post

Social Media Image Generator is a script to generate social media images for blog posts. It utilizes Puppeteer for running a headless Chromium to take a screenshot of a temporary created HTML file, which is generated with your posts Frontmatter information and a referenced photograph.


For more information visit:

Github or Open in VSCode

Hexo Console Webmention 1 Post

Hexo Console Webmention is a console command for Hexo to send Webmentions from your blog to others. It utilizes the work of Remy Sharp, who wrote a JS library to parse a URL for sending Webmentions to linked Blogs. It is mainly used at webmention.app



For more information visit:

Github NPM Insights or Open in VSCode
